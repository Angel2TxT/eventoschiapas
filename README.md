# Eventos Chiapas

Sitio web estático básico para la práctica de administración de proyectos de TI con Git y GitHub.

## Descripción

Proyecto individual basado en la consigna de la asignatura. El sitio incluye:

- `index.html` como portada
- `eventos.html` con catálogo básico de eventos
- `contacto.html` con formulario y datos de contacto
- estilos compartidos en `css/style.css`
- comportamiento simple en `js/script.js`

## Objetivo

Demostrar el uso de Git y GitHub en un flujo de trabajo básico con estructura ordenada, documentación y publicación en GitHub Pages.
